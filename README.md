# Инструкция по запуску проекта

Чтобы запустить приложение, выполните следующие шаги:

### **1. Запуск backend-сервера**

Перейдите в папку `backend`:

```bash
cd backend
```

Установите зависимости:

```bash
npm install
```

Запустите сервер:

```bash
node src/index.js
```

Убедитесь, что в терминале появилось сообщение:

```
Backend running on port 3001
```

### **2. Запуск frontend-клиента**

Откройте второй терминал, перейдите в папку `frontend`:

```bash
cd frontend
```

Установите зависимости:

```bash
npm install
```

Запустите клиент:

```bash
npm start
```

После этого проект успешно запустится.

# О проекте

За основу идеи была взята картотека жителей вымышленного города **А**.
Приложение позволяет:

* Просматривать список всех жителей
* Смотреть подробную статистику
* Создавать новых жителей
* Отправлять форму обратной связи

На главной странице доступны:

* Поиск жителя по имени
* Удобные фильтры

---

# Важная информация

Все данные о жителях были сгенерированы при помощи **faker**, поэтому возможны логические несостыковки.
Например:

* У жителя могут быть указаны супруга и дети
* Но в карточке детей может не отображаться эта же супруга как родитель

Это нормальное следствие рандомной генерации данных.
